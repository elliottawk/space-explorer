NASA API PROJECT

Explore real space data through an interactive website.

Space Explorer uses NASA's public API to display near-Earth asteroid data with a clear loading spinner, filter controls, and responsive cards.

View Asteroids View Missions

Project Features

  • NASA API integration with API key
  • Loading spinner during API calls
  • Client-side filter for asteroid results
  • Error, empty, and success states
  • 15 curated real space missions
